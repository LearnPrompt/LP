# 🟢 简介

随着 ChatGPT 的爆火，越来越多的人开始关注并尝试 AI 相关的应用。而图片生成就是其中一个备受瞩目的领域。目前已经有许多图像生成工具，如 Midjourney，Stable Diffusion，DALL-E，Firefly等。本课程主要是以 Midjourney 为主，让你可以快速了解并学会使用文生图工具。
